Boolean Attribute

Boolean attributes are attributes that are evaluated either true or false. A number of attributes are boolean attributes. The presence of a boolean attribute on an element represents the true value, and the absence of the attribute represents the false value.

HTML boolean attributes list:

  • async
  • autocomplete
  • autoplay
  • checked
  • controls
  • default
  • loop
  • required
  • selected
  • muted
  • scrolling, etc.
